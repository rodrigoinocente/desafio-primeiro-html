O que são as tags:

As tags em HTML são elementos usados para marcar e estruturar o conteúdo em uma página da web. Elas consistem em texto dentro de ângulos (< e>) e são usadas para definir como o conteúdo deve ser exibido e interpretado pelo navegador. Cada tag possui um propósito específico, como a formatação de texto, a criação de links, a exibição de imagens e a organização de listas. As tags podem conter atributos que fornecem informações adicionais sobre o elemento. O HTML utiliza uma série de tags predefinidas para criar a estrutura e o design das páginas da web, e os desenvolvedores web podem combiná-las para criar páginas interativas e visualmente atraentes.

<h1>

As tags de cabeçalho <h1> a <h6> são usadas para criar títulos e subtítulos com diferentes níveis de importância. <h1> é o mais importante e <h6> é o menos importante na hierarquia de títulos. Eles são usados para organizar o conteúdo da página, com <h1> representando o título principal e <h6> sendo usado para títulos de menor importância. A formatação visual pode ser ajustada com CSS.

<p>

Server para criar parágrafos de texto. Ela é usada para separar e estruturar o texto em blocos distintos, tornando a leitura e organização do conteúdo mais clara em uma página da web. O conteúdo dentro da tag <p> é exibido com um espaço em branco acima e abaixo dele por padrão, criando assim parágrafos facilmente legíveis.

<small>

A funcionalidade da tag <small> é para diminuir o tamanho do texto, tornando-o visualmente menor do que o texto circundante. É frequentemente usado para indicar que um texto é menos importante ou para exibir informações auxiliares, como notas de rodapé, datas de copyright ou detalhes de direitos autorais.

<i>

A tag <i> se faz necessário quando queremos aplicar ênfase ou itálico ao texto. Geralmente não altera significativamente o significado do texto, mas é usado para indicar uma ênfase suave. Por exemplo, pode ser usado para títulos de obras (livros, filmes) ou termos estrangeiros.

<ol>

Essa ferramenta poderosa é utilizada para criar listas ordenadas, onde os itens são automaticamente numerados em ordem sequencial. Essas listas são úteis para apresentar informações de maneira estruturada e lógica, especialmente quando a ordem dos itens é importante. Cada item da lista é definido com a tag <li>, permitindo a criação de listas de passos, rankings e muito mais.

<li>

Quando queremos criar itens em listas, sejam elas ordenadas (com <ol>) ou não ordenadas (com <ul>). Cada <li> define um item individual na lista, permitindo a organização de informações em uma sequência ou grupo.

<ul>

Usada para criar listas não ordenadas, nas quais os itens não são numerados sequencialmente. Ela é útil para organizar informações em uma ordem que não possui importância sequencial, como listas de itens sem uma ordem específica. Cada item da lista é definido com a tag <li>.

<a>

Para criar links em uma página da web essa tag permite que você vincule o conteúdo a outras páginas da web, arquivos, ou recursos online, tornando a navegação entre páginas e sites mais fácil. Você pode definir o destino do link usando o atributo "href".

<hr/>

Quando se faz necessário criar uma linha horizontal ou uma "régua" em uma página da web. Ela é frequentemente usada para separar visualmente o conteúdo, como divisões em seções ou tópicos, sem criar uma quebra de parágrafo. O <hr> não requer uma tag de fechamento e é autocontido.

<sup>

Sua funcionalidade é para criar texto sobrescrito, o que significa que o texto é exibido ligeiramente acima da linha de base normal. Geralmente é usado para representar expoentes, como em fórmulas matemáticas e para exibir símbolos como graus (°).

<blockquote>

Em HTML é usada para citar blocos de texto de outra fonte em um documento. Ela é frequentemente usada para destacar citações ou trechos de texto longos e, por padrão, adiciona um recuo à esquerda para diferenciar o texto citado do restante do conteúdo.

<img/>

Utilizada para mostrar imagens em uma página da web. Ela não tem uma tag de fechamento e requer o atributo "src"para especificar o caminho da imagem.

<br/>

Usada para criar uma quebra de linha em uma página da web. Ela não possui um tag de fechamento, e quando você a insere em seu código HTML, faz com que o texto ou conteúdo subsequente apareça em uma nova linha. É útil quando você deseja criar espaços ou separar linhas de texto, como em um poema ou endereço.
